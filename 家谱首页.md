# 杜氏家谱

## 家族历史

明洪武元年（约公元1370年）奉诏迁居**开封府繁塔侧云寄桥**，后逐渐流居于杞县西寨、柿国万寨、祥符仇楼、杜良等地。

## 如何维护

### 新增成员
1. 复制 [[Templates/人物模板]] 到 `People/` 文件夹，重命名为姓名
2. 填写 YAML 中的关系字段，用 `[[对方姓名]]` 双向链接
3. 在父母/配偶/子女对应页面同步添加反向链接
4. 在 [[家族关系图]] 中添加新节点和连线
